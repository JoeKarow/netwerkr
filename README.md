## About

> Networking is hard work, especially for those with a poor working memory.
> netwerkr seeks to help keep track of all of those amazing new connections.

## Getting Started

### Prerequisites

- [NodeJS >= 16](https://nodejs.dev/)
- [pnpm](https://pnpm.io/)

### Installation

  ```bash
pnpm install
  ```

## Usage

```bash
pnpm dev
```
